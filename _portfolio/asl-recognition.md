## Overview
GUI application using CNN classifier and YOLO object detection to verify correct American Sign Language letter signing. Developed at Hack the Northeast hackathon.

## Technical Details
- **Classification**: Convolutional Neural Network
- **Object Detection**: YOLO
- **Framework**: TensorFlow
- **GUI**: tkinter
- **Event**: Hack the Northeast

## Features
- Real-time hand detection using YOLO
- ASL letter classification with CNN
- Visual feedback for learning
- Interactive GUI for accessibility
